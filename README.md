Overview
Playlist Skins 3.0 is a web-based music player that places a Spotify playlist or other media player inside customizable boombox skins.
Each skin is a PNG image designed to look like a real stereo, radio, or music device. The playlist appears inside the display window of the device, making it look like the device itself is playing the music.
Instead of embedding a plain playlist box, websites can display a visual music player.

Boombox Layout System
The boombox player uses absolute positioning to place the playlist window inside the boombox image.
Example structure:
div class="boombox"
image src="skins/red-sharp.png"
div class="playlist-window"
iframe src="SPOTIFY_PLAYLIST_URL"
end div
end div
The playlist window sits directly over the display area of the boombox.
Example CSS Layout
.boombox
position relative
width 900px
.boombox img
width 100 percent
.playlist-window
position absolute
top 340px
left 300px
width 400px
height 220px
overflow hidden
This allows precise placement of the player within the boombox display.
