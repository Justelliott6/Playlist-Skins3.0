# Playlist Skins 3.0
Boombox Music Player Platform

Developer Guide for the Boombox Playlist App
Project Owner: Elliott Palmer

---

## Project Overview

Playlist Skins 3.0 is a web-based music player interface that places a Spotify playlist
(or other media embeds) inside customizable boombox skins.

Each skin is a PNG image styled to look like a vintage stereo, radio, or music device.

Users can:

• Display Spotify playlists inside themed boombox players  
• Switch between multiple visual skins  
• Embed Facebook streams or radio players  
• Generate embed code for websites  
• Install the player as a PWA web app  

---

## Core Concept

The player works using three simple components:

Boombox Skin (PNG Image)  
+ Embedded Media Player (Spotify / Video / Radio)  
+ HTML + CSS Layout  

= Boombox Music Player

The image itself is **not generated by code**.

The code simply places the playable media inside the display window
of the boombox image.

## Boombox Layout System

The playlist window is placed over the display screen using CSS positioning.

Example layout:

div class="boombox"  
img src="skins/red-sharp.png"

div class="playlist-window"  
iframe src="SPOTIFY_PLAYLIST_URL"

---

## Spotify Playlist Embed

Spotify provides official playlist embeds.

Example:

iframe  
src="https://open.spotify.com/embed/playlist/PLAYLIST_ID"

---

## Facebook Integration

Facebook video streams can also be embedded.

Example:

div class="fb-video"
data-href="FACEBOOK_VIDEO_URL"

---

## Skin System

Each skin is a PNG image stored in the skins folder.

Examples:

skins/red-sharp.png  
skins/spotify-green.png  
skins/facebook-blue.png  
skins/purple-rabbit.png

New skins can be added at any time without changing the core code.
